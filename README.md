# Installation

Prerquisite : install [git](https://git-scm.com/) and [node](https://nodejs.org/en/).

Then in your terminal run :

```
git clone https://github.com/medialab/portic-datasprint-2021
cd portic-datasprint-2021
chmod +x install.sh
./install.sh
```
